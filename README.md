# Expedición Espacial / Gigathon_2026

## Descripción

Expedición Espacial es un juego desarrollado en Python en el que el jugador controla a un astronauta que explora un planeta desconocido. El objetivo principal es encontrar una base alienígena antes de quedarse sin energía o sin movimientos.

Durante la exploración podrán aparecer objetos alienígenas con efectos positivos o negativos, además de eventos aleatorios que pueden ayudar o dificultar la misión.

---

## Características

* Generación aleatoria de la base alienígena.
* Cinco objetos alienígenas con distintos efectos sobre la energía.
* Eventos aleatorios:

  * Misil de energía (+500 de energía).
  * Teletransporte a una posición aleatoria.
* Compra de movimientos usando energía.
* Compra de una pista sobre la ubicación de la base.
* Informe final con el resumen completo de la expedición.
* Validación de todas las entradas del usuario.

---

## Requisitos

* Python 3.12.10
* No es necesario instalar librerías adicionales, únicamente se utiliza el módulo `random`, incluido en Python.

---

## Cómo ejecutar el juego

1. Descarga el archivo del proyecto.
2. Abre una terminal en la carpeta donde se encuentra.
3. Ejecuta:

```bash
python main.py
```

---

## Cómo jugar

1. Introduce los datos de la expedición.
2. Elige el tamaño del mapa y el número de movimientos.
3. Explora el planeta moviéndote por el mapa.
4. Gestiona correctamente la energía y los movimientos disponibles.
5. Encuentra la base alienígena antes de que termine la expedición.
